各種分類機のファイル、.pklファイルはトレーニング済みのモデル。
分類に用いるパラメーターはprepare()の中で作る。
PCA（主成分分析）は入力行列が約200×60次元で大きいため、これらの分類機用に次元を減らすために試したけど精度は上がらなかった、コード内のフラグでon/off切り替え。
トレーニング済みのモデルを使うときはclf = joblib.load('svmModel.pkl’)でできる。
毎回プログラムを走らせると.pklファイルを探す為、新しいモデルをトレーニングしたいときは、clf = joblib.load('svmModel.pkl____’)ここの引数にあるモデルファイルを名前を変えるなどして読み込まないようにする。モデルファイルが見つからなかった場合新しいモデルを作り始める。
